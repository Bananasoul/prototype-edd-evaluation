# École du Dos — Prototype d'évaluation anonyme

Interface d'évaluation anonyme pour l'École du Dos (Rückenschule) — Hôpital Saint-Nicolas Eupen.

## Questionnaires inclus

- **HAD-S** — Hospital Anxiety and Depression Scale
- **ODI** — Oswestry Disability Index 2.1a
- **TSK-17** — Tampa Scale of Kinesiophobia

## Langues

- 🇫🇷 Français (versions validées)
- 🇩🇪 Deutsch (validierte Versionen)

## Fonctionnement

1. Le patient sélectionne la langue
2. Il répond aux 3 questionnaires de façon anonyme
3. Un écran de remerciement s'affiche
4. Le thérapeute accède aux résultats via un bouton discret (dashboard visuel)

## Déploiement

### Vercel
```bash
# Installer Vercel CLI
npm i -g vercel

# Déployer
vercel
```

### Développement local
```bash
npx serve .
```
